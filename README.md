# Telecom Call Centre Dashboard

## Overview
This project presents an interactive Call Centre Analytics Dashboard for a telecommunications company. The dashboard provides insights into customer service performance, operational efficiency, workforce productivity, and revenue generation during FY2024.

## Key KPIs
- Total Calls: 219,816
- Calls Answered: 203,161
- Answer Rate: 92.4%
- Abandonment Rate: 7.6%
- First Call Resolution (FCR): 76.4%
- Customer Satisfaction (CSAT): 4.08/5
- Service Level Agreement (SLA): 82.1%
- Agent Occupancy: 83.3%
- Total Revenue: MWK 137.0 Million
- Escalations: 2,149
- Complaints: 1,274

## Dashboard Features
- Call Volume Analysis
- Service Quality Monitoring
- Customer Satisfaction Tracking
- Agent Performance Monitoring
- Revenue Analysis
- Escalation and Complaint Tracking
- Monthly KPI Trends

## Key Insights
- Strong answer rate of 92.4%.
- High customer satisfaction score of 4.08/5.
- September generated the highest revenue.
- November recorded the highest call volume.
- Efficient workforce utilization with occupancy above 83%.

## Tools Used
- Microsoft Excel
- Data Visualization
- KPI Reporting
- Business Intelligence

## Business Impact
The dashboard supports telecom management in monitoring customer experience, operational efficiency, and revenue performance, enabling evidence-based decision-making.
